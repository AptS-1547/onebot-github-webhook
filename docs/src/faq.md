## 常见问题

**Q: Webhook 触发但 QQ 未收到消息怎么办？**

A: 请检查以下几点：

1. OneBot 地址和 token 是否正确
2. OneBot 实现是否支持 WebSocket/HTTP 连接
3. QQ 机器人是否在目标群/好友中
4. 查看日志获取详细错误信息

**Q: 如何支持多个 QQ 机器人？**

A: 本程序暂时不支持推送到多个 QQ 机器人
